Overview

Wireless Sensor Network, often abbreviated as “WSN”, is an emerging and fast-growing technology. One network of this type consists of a large number of sensor nodes, also called “motes”, spatially distributed.

Security Issues

Wireless Sensor Networks are vulnerable to various attacks like any other conventional network, but its limited resource characteristics and unique application features requires some extra security requirements including the typical network requirements.

View details »

Layers

The main goal of security is to provide protection services from all kinds of attacks. Achieving this is difficult, but the complex systems designed with this purpose must rely on a set of requirements that should be able to guarantee scalability, efficiency and reliability. There are are several properties that the foundation of a secure wireless sensor network should be built on.

View details »

Data Aggregation

Sensor nodes present several limitations, mostly due to their constrained power capabilities1. Amongst the features of sensor nodes, data communication is notably the most power consuming. Hence, it is important to minimise the amount of data transiting through the Wireless Sensor Network and optimise the overall bandwidth utilisation by preventing overwhelming amounts to traffic.
